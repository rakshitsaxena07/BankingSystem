# Bank Account OOP Project (C++)

## Problem Statement
Create a simple bank account system using Object-Oriented Programming in C++.
The system should support:
- Creating a bank account
- Depositing money
- Withdrawing money
- Checking balance

## OOP Concepts Used
- Class and Object
- Encapsulation
- Constructors


## How to Run
g++ main.cpp BankAccount.cpp -o bank
./bank
